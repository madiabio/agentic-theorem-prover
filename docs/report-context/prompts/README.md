# Prompt Context Entries

This directory stores one report-context entry per Markdown file. Use it for
selected prompts, decisions, experiments, or agent instructions that are likely
to help write the final report.

`AGENTS.md` defines the required body for each entry. This README only documents
where the files live and how to name them.

Name each entry with local repo time and a short slug:

```text
YYYYMMDD-HHMMSS-short-title.md
```

Do not save routine prompts, debugging chatter, secrets, API keys, or private
credentials.
